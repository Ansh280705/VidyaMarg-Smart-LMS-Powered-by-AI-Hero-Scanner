# AI-Powered Student Support System

This project is a complete student support platform that includes scanning tools, an AI chatbot, career guidance, and performance reporting.  
It is built using **Node.js**, **Express**, **MongoDB**, **EJS**, and **TailwindCSS**.

---

## Features

### User System

- Secure user signup
- Password hashing using **bcrypt**
- JWT-based authentication

### AI Chatbot

- Answers student questions
- Provides academic guidance
- Works as a personal assistant

### Normal Scanner

- Scans documents, text, and QR codes
- Performs basic OCR-style reading

### Hero Scanner

- Advanced scanner
- Detects patterns and structures
- More accurate and faster

### AI Career Coach

- Suggests personalized career paths
- Matches skills to career options
- Helps students grow with proper guidance

### Progress Report System

- Shows student performance
- Tracks improvements
- Generates clean reports

---

## Technologies Used

# Backend Technologies

- Node.js – Backend runtime

- Express.js – API routing and server logic

- MongoDB – Database for users, reports, and scan results

- Mongoose – Data modeling and schema management

# Frontend Technologies

- EJS – Dynamic server-side templating

- TailwindCSS – Modern responsive styling

- Authentication & Security

- JSON Web Tokens (JWT) – Secure authentication

- Bcrypt – Password hashing

- Cookie Parser – Managing tokens and user sessions

# AI and Intelligent Modules

- AI Chatbot Engine – Response generation and support

- Normal Scanner Module – Basic OCR-style data extraction

- Hero Scanner Module – Advanced document processing and pattern detection

- AI Career Coach Engine – Career recommendation logic based on user data

# Development Tools

- Nodemon – Auto-restart server during development

- Dotenv – Environment variable management

- Git & GitHub – Version control and project hosting
